# PSX Side Panel Recreation

[![CC BY-NC-SA 4.0][cc-by-nc-sa-shield]][cc-by-nc-sa]

These are recreations of the plastic side panels found on the Sony PSX DVR.
Some small changes were made to improve durability and 3D printing friendliness. This includes removing the thin feature on the interior front, and making more solid the mating features.
